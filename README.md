# Kargo

[![GitHub license](https://img.shields.io/badge/license-Apache%20License%202.0-blue.svg?style=flat)](https://www.apache.org/licenses/LICENSE-2.0)

Kargo is a build tool derived from JetBrains Amper, developed with a different vision and priorities.

## Why Kargo?

Kargo explores ideas and design decisions that diverge from Amper, including:

- A more explicit and deterministic build model
- Native-first workflows
- Reduced implicit behavior
- Long-term architectural evolution

## Project Status

Early stage. APIs and behavior may change.

## Relationship with Amper

Kargo started as a fork of JetBrains Amper.
It is not affiliated with JetBrains.
Compatibility with Amper is not guaranteed.
